Sea Lantern（海晶灯）新手使用教程

文档版本：0.0.1

状态：持续更新中

项目概述

Sea Lantern（海晶灯）是一个专为Minecraft服务器设计的管理工具。它使用Tauri 2 + Rust + Vue 3技术栈开发，具有启动快、体积小、内存占用少的特点。

项目功能

简单启动

导入一个服务端 JAR 文件，选一个 Java，点启动。就这么简单。

实时控制台

控制台实时看日志，直接输命令

图形化配置

server.properties 图形化编辑，不用手改文件

玩家管理

白名单、封禁、OP 一键管理

自动保存

关软件的时候自动帮你停服务器，不会丢存档

自动更新

检查更新，一键下载新版本

系统要求

作为用户使用

如果你只是想使用Sea Lantern来管理Minecraft服务器，你只需要下载发布版本即可，无需安装额外软件。

支持的操作系统版本

  • Windows：Windows 10 (版本 1909 及以上) 或 Windows 11
  • macOS：macOS 10.15 (Catalina) 及以上
  • Linux：支持现代Linux发行版（Ubuntu 20.04+, Fedora 34+, Debian 11+ 等）

Windows版本兼容性说明

Sea Lantern基于现代Tauri框架开发，需要较新版本的Windows系统组件支持。因此，官方支持以下Windows版本：

  • Windows 10：版本 1909 (10.0.18363) 及以上（推荐版本 20H2 或更高）
  • Windows 11：所有版本

不支持的Windows版本：Windows 7, Windows 8, Windows 8.1

作为开发者开发

如果你希望参与开发或自定义编译，需要：

  • Node.js 20+
  • Rust 1.70+
  • 对于Windows开发：Windows 10/11开发环境

快速开始

下载与安装

  1. 下载Sea Lantern
    • 访问Sea Lantern的发布页面（Gitee）
    • 下载最新版本的安装包
    • 根据你的操作系统选择相应的版本（Windows、macOS、Linux）
  2. 安装Sea Lantern
    • Windows用户：运行下载的.exe安装文件
    • macOS用户：运行.dmg文件
    • Linux用户：运行.AppImage文件或使用包管理器安装

获取服务器启动核心（JAR文件）

什么是服务器JAR文件？

服务器JAR文件（Java Archive）是Minecraft服务器的启动文件，它包含了运行服务器所需的所有代码和资源。根据你的需求，可以选择不同类型的服务器JAR文件：

  • 原版服务器（Vanilla）：Mojang官方发布的纯净服务器，没有任何修改
  • Spigot服务器：基于原版的高性能服务器，支持插件
  • Paper服务器：基于Spigot的高性能服务器，优化了性能和功能
  • Bukkit服务器：较早的插件API服务器（现已不推荐使用）
  • Forge服务器：支持模组（Mod）的服务器
  • Fabric服务器：轻量级模组API服务器

获取原版服务器JAR

  1. 访问 Minecraft官网服务器页面
  2. 下载服务器JAR文件（文件名通常为server.jar）
  3. 注意：需要Minecraft账户才能下载

获取高性能服务器JAR（Spigot或Paper）

  1. 获取Paper服务器（推荐）：
    • 访问 PaperMC官方网站
    • 选择适合的Minecraft版本
    • 点击"Download"按钮下载对应的JAR文件
  2. 获取Spigot服务器：
    • 访问 Bukkit官方网站（注意：由于Spigot不直接提供预编译版本，需要使用BuildTools编译）
    • 由于Spigot不直接提供预编译版本，需要使用BuildTools编译
    • 下载BuildTools.jar文件并按照官方指南编译

获取模组服务器JAR（Forge或Fabric）

  1. 获取Forge服务器：
    • 访问 Forge官网
    • 选择适合的Minecraft版本
    • 下载"Installer"版本的JAR文件
    • 双击运行安装器来设置服务器
  2. 获取Fabric服务器：
    • 访问 Fabric官网
    • 下载Fabric安装器
    • 运行安装器来设置服务器

注意事项

  • 确保选择的服务器JAR与你计划使用的客户端版本兼容
  • Paper和Spigot服务器在第一次运行时会自动生成EULA文件，需要同意才能使用
  • 模组服务器（Forge/Fabric）需要额外安装模组才能发挥功能
  • 选择服务器时要考虑性能需求、插件兼容性和易用性

创建第一个Minecraft服务器

  1. 启动Sea Lantern
    • 双击桌面图标启动Sea Lantern
    • 等待应用加载完成（你会看到启动画面）
  2. 导入服务器
    • 点击侧边栏的"创建服务器"选项
    • 点击"导入服务器"按钮
    • 选择你的Minecraft服务器JAR文件（如server.jar）
    • 选择服务器名称和保存位置
    • 点击"创建服务器"
  3. 配置Java环境
    • Sea Lantern会自动检测系统中的Java环境
    • 如果自动检测失败，可以手动指定Java路径
    • 点击"Java检测"按钮，选择合适的Java版本（推荐使用与你的Minecraft版本兼容的Java版本）
  4. 配置服务器参数
    • 点击"配置编辑"选项卡
    • 你可以在这里图形化编辑server.properties文件
    • 修改服务器名称、游戏模式、难度等参数
    • 点击"保存配置"按钮保存更改

启动和管理服务器

启动服务器

  • 回到首页，你会看到刚刚创建的服务器
  • 点击"启动"按钮启动服务器
  • 等待服务器启动完成（控制台会显示启动日志）

使用控制台

  • 点击"控制台"选项卡
  • 你可以在这里查看服务器日志
  • 在底部输入框中输入Minecraft命令（如/help、/op username等）
  • 控制台支持命令历史（上下箭头键）和Tab补全

管理玩家

  • 点击"玩家管理"选项卡
  • 你可以管理白名单、封禁列表和OP列表
  • 点击"+"按钮添加玩家到相应列表
  • 点击"-"按钮移除玩家
  • 也可以直接在控制台输入相关命令

服务器配置详解

基本设置

  • 服务器名称：服务器的显示名称
  • 服务器端口：服务器监听的端口，默认为25565
  • 最大玩家数：服务器允许的最大在线玩家数
  • 游戏模式：生存模式、创造模式、冒险模式或旁观模式
  • 难度：和平、简单、普通或困难

游戏设置

  • PVP：是否允许玩家之间战斗
  • 生成建筑：是否在世界中生成要塞、村庄等建筑
  • 生成矿石：是否生成矿物
  • 世界类型：默认世界生成类型

网络设置

  • 最大构建宽度：客户端可以达到的最大构建宽度
  • 最大跟踪距离：服务器向客户端发送数据的最大距离
  • 在线模式：是否验证玩家的Minecraft账户

获取和安装服务器插件

什么是服务器插件？

服务器插件是为Minecraft服务器添加额外功能的程序。它们可以添加新的命令、修改游戏机制、提供管理工具等。插件通常以.jar文件形式分发。

支持插件的服务器类型

  • Paper服务器：支持所有Bukkit/Spigot插件，性能优秀
  • Spigot服务器：支持Bukkit插件，功能丰富
  • Bukkit服务器：最早的插件API（现已不推荐使用）
  • Purpur服务器：基于Paper的分支，提供更多配置选项

注意：原版服务器、Forge和Fabric不支持传统插件，需要使用模组。

获取插件的途径

  1. SpigotMC资源站：
    • 访问 SpigotMC资源页面
    • 注册账户（免费）
    • 搜索和下载所需的插件
    • 注意检查插件的兼容性版本和评分
  2. Polymart资源站：
    • 访问 Polymart资源页面
    • 类似于SpigotMC，提供大量插件
    • 界面更现代化
  3. GitHub开源项目：
    • 搜索GitHub上的开源插件项目
    • 通常在Releases部分提供下载
    • 可以查看源代码和提交的Issue
  4. Planet Minecraft：
    • 访问 Planet Minecraft资源页面
    • 提供多种类型的Minecraft资源

安装插件的步骤

  1. 准备工作：
    • 确保服务器已停止（如果正在运行）
    • 找到服务器目录中的plugins文件夹
    • 如果没有plugins文件夹，可手动创建一个
  2. 放置插件文件：
    • 将下载的插件JAR文件复制到plugins文件夹中
    • 确保插件文件名不包含特殊字符或空格
    • 每次只安装一个插件以避免冲突
  3. 配置插件：
    • 启动服务器，插件会自动生成配置文件
    • 在plugins文件夹中找到插件的配置文件夹
    • 根据需要编辑配置文件（通常是config.yml）
  4. 重启服务器：
    • 重启服务器以使插件生效
    • 检查控制台日志确认插件加载成功

使用Sea Lantern管理插件

虽然Sea Lantern目前没有专门的插件管理界面，但你可以通过以下方式使用它来管理插件：

  • 启动服务器：使用Sea Lantern启动服务器时会自动加载插件
  • 查看日志：通过控制台查看插件是否正确加载
  • 执行命令：某些插件需要在启动后执行特定命令来完成安装
  • 查看插件列表：使用/plugins命令查看已加载的插件
  • 停止服务器：使用Sea Lantern停止服务器以安全地修改插件

常见插件类型

  • 管理类插件：
    • EssentialsX：提供基础管理命令（传送、家、经济等）
    • WorldEdit：强大的世界编辑工具
    • WorldGuard：区域保护和权限管理
  • 游戏增强类插件：
    • CoreProtect：区块和玩家操作日志记录
    • Multiverse-Core：多世界管理
    • DiscordSRV：服务器与Discord的集成
  • 经济系统插件：
    • Vault：经济API（需要配合经济插件使用）
    • Shopkeepers：商人NPC系统

插件兼容性注意事项

  • 确保插件与你的服务器版本兼容
  • 检查插件之间的依赖关系
  • 避免安装功能重复的插件
  • 定期更新插件以获得新功能和安全修复
  • 备份服务器数据后再安装新插件

高级功能

自动更新

Sea Lantern支持自动检查更新功能。在"关于"页面中点击"检查更新"可以手动检查新版本。如果有新版本，会显示更新日志并提供下载链接。

应用设置

  1. 自动停止服务器

    在设置页面可以启用"关闭软件时自动停止服务器"。这样可以防止在关闭Sea Lantern时丢失存档。

  2. 内存设置

    可以设置默认的Java堆内存大小。根据你的系统配置和服务器需求调整。

  3. JVM参数

    可以自定义JVM启动参数。高级用户可以优化服务器性能。

故障排除

无法启动服务器

  1. 检查Java路径是否正确
  2. 确保服务器JAR文件完整且兼容
  3. 检查端口是否被其他程序占用
  4. 查看控制台日志以获取错误信息

控制台无响应

  1. 检查服务器是否正常运行
  2. 确认网络连接正常
  3. 重启Sea Lantern应用

无法连接服务器

  1. 检查服务器是否已启动
  2. 确认端口设置正确
  3. 检查防火墙设置
  4. 确保服务器设置为允许外部连接

开发者指南

如果你想参与Sea Lantern的开发：

1. 克隆项目

git clone https://gitee.com/fps_z/SeaLantern.git
cd SeaLantern

2. 安装依赖

npm install

3. 启动开发模式

npm run tauri dev

4. 构建发布版

npm run tauri build

构建产物位于 src-tauri/target/release/bundle/ 目录中。

技术架构

Sea Lantern采用前后端分离架构：

  • 前端：Vue 3 + TypeScript + Vite + Pinia
  • 后端：Rust + Tauri 2
  • 样式：纯 CSS
  • 通信：Tauri invoke（前端调用Rust函数）

常见问题解答

问题 答案
Sea Lantern需要什么权限？ Sea Lantern需要访问文件系统、网络和执行程序的权限，这些是管理Minecraft服务器所必需的。
如何备份服务器？ 目前需要手动备份服务器目录，未来版本将集成备份功能。
Sea Lantern是否安全？ Sea Lantern是开源软件，代码完全透明。它不会连接到任何第三方服务器，所有数据都保存在本地。
支持哪些Minecraft版本？ Sea Lantern支持所有标准的Minecraft服务器版本，只需使用对应版本的服务器JAR文件。
可以在Windows 7/8/8.1上运行Sea Lantern吗？ Sea Lantern官方不支持Windows 7/8/8.1系统。默认情况下Sea Lantern不支持原生Windows 7运行，但可以通过安装VxKex扩展内核以补全所需运行库来运行。详情请参考"旧版Windows运行方案"章节。

旧版Windows运行方案

关于Sea Lantern在Win10以下系统运行：默认情况下Sea Lantern不支持原生Windows 7运行，通过安装VxKex 扩展内核以补全所需运行库来运行，这对已然处于旧系统的用户提供了一个解决方案（注：此方式同样适用于一些无法在老系统上正常运行的软件）

具体步骤

  1. 获取Sea Lantern软件压缩包：
    • 前往Sea Lantern官网下载Windows压缩包
    • 解压完成不要打开Sea Lantern
  2. 下载并安装VxKex 扩展内核：
    • 此处提供的是国内的一个分支
    • VxKex-NEXT：github.com/YuZhouRen86/VxKex-NEXT
    • 蓝奏云：geekerz.lanzoue.com/b0ra5olfi
    • 百度网盘：pan.baidu.com/s/1w0JUk4JpoKiZQVi2LMoqwA?pwd=0000
    • 提取码：0000
    • 下载完成无脑安装即可
  3. 给Sea Lantern启用VxKex：
    • 右键Sea Lantern主程序，选择属性，选择[VxKex]
    • 勾选[为此程序启用VxKex NEXT]和[报告其他版本的Windows]，应用，确定
    • 此时Sea Lantern即可在Windows 7上正常运行
    • 除联机外，其他功能均正常

结语

此方法通过VxKex扩展内核为旧版Windows系统补全运行库，以支持现代软件运行。虽然Sea Lantern在旧版系统上大部分功能可以正常运行，但可能某些高级功能会受限。

总结

Sea Lantern是一个功能强大且易于使用的Minecraft服务器管理工具。通过本教程，你应该已经学会了如何安装、配置和管理Minecraft服务器。如果遇到问题，可以查看控制台日志获取更多信息，或者在项目社区中寻求帮助。

重要警告与免责声明

关于在旧版Windows系统上运行Sea Lantern的警告：

  • 使用非官方方法（如VxKex扩展内核）在Windows 7/8/8.1等旧版系统上运行Sea Lantern并不保证能正常工作，可能会出现各种错误和兼容性问题。
  • 开发者不会接受任何通过此方法运行软件所产生的错误反馈，也不会对由此产生的问题进行修复。
  • 使用此类方法可能存在系统安全风险，用户需自行承担相关责任。
  • 强烈建议升级到受支持的Windows版本（Windows 10/11）以获得最佳体验和完整功能支持。
  • Sea Lantern的开发和测试均基于现代Windows系统，对旧版系统的兼容性不在官方支持范围内。
